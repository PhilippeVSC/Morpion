Le Morpion est un jeu de plateau classique, également connu sous le nom de Tic-Tac-Toe, réalisé en utilisant les technologies web standards telles que HTML, CSS et JavaScript. Le jeu est basé sur une grille dynamique générée en JavaScript, où les joueurs peuvent placer leur symbole (X ou O) sur une case en cliquant dessus. Le code vérifie ensuite automatiquement les lignes verticales, horizontales et diagonales pour détecter si l'un des joueurs a remporté la partie. En cas d'égalité, le code compte le nombre de tours et déclare une égalité si aucun joueur n'a gagné après 9 tours.
